# agenticbook-starter

**A complete example world model for *Agentic Book Writing: The Smart Way*.**

This repository contains a fully filled-in Short-flavor world model for a fictional book project: *First Light: A Beginner's Guide to Night Sky Photography*. Every document demonstrates a real-world-model pattern from the methodology — with inline comments explaining *why* each field is written the way it is.

Use this as:
- A **reference** while reading the book (see how the docs connect)
- A **template** to copy and overwrite for your own project
- A **teaching example** for the concepts in Chapters 3–8

## How to use this

### Option A: Start from the example (fastest)
1. Clone this repo: `git clone git@github.com:RobLe3/agenticbook-starter.git mybook`
2. Read `docs/00_index.md` to understand the structure
3. Open each doc and replace the *First Light* content with your own project
4. You have a complete Short-flavor world model skeleton in under 2 hours

### Option B: Start from blank templates (most flexible)
1. Clone the repo
2. Copy the files from `blank/` to a new `docs/` folder in your project
3. Follow the labs in *Agentic Book Writing: The Smart Way* to fill them in
   (Appendix E contains Lab E — a step-by-step setup sequence for this kit)

### Option C: Reference only
Browse the docs while reading Chapters 3–8. Each doc maps to a specific chapter:

## The example project

**Title:** *First Light: A Beginner's Guide to Night Sky Photography*
**Flavor:** Short (target: 22,000–28,000 words)
**Genre:** Non-fiction how-to / beginner guide

This fictional book was chosen as the example because:
- It requires a real technical glossary (demonstrating Doc 04)
- It has distinct reader personas with different needs (demonstrating Doc 02)
- It has a clear chapter structure a beginner can plan in an afternoon (demonstrating Doc 03)
- It is completely unrelated to AI or writing — demonstrating that the methodology works for **any** book topic

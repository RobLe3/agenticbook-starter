## C. Quality Gate

- [ ] All chapters scored using the 10-dimension rubric

- [ ] Ship-it signals checked: 3 of 5 must be true (see Doc 04 → Ship-It Signals)

## D. Metadata and Packaging Readiness

- [ ] Export format tested: PDF (print layout) and/or EPUB (ebook)

- [ ] Author bio complete and current (no placeholder text)
- [ ] Categories and keywords filed correctly for platform (Photography > Night Photography)
- [ ] Final formatting pass: no orphaned headers, broken figures, or missing images

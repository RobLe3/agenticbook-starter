# tools/ — Versioned Tool Guides

Six versioned tool guides for the tools referenced throughout *Agentic Book Writing: The Smart Way*. This folder contains the expanded, dated guides that replaced the in-book appendix content.

## Why here instead of in the book

Tool installation instructions go stale. A command that worked in 2025 may not work in 2026. The book's Appendix B provides the stable methodology; this folder provides the current versioned commands — updated with each tagged repo release.

**When the book and the repo diverge on a command:** use the repo version. It is versioned and dated. The book version is illustrative.

## Part I: Canon Hierarchy

Technical claims in this book have three authority levels:

| Level | Source | Examples |
|-------|--------|---------|
| **Level 1 — Fermentation science** | Peer-reviewed research, established food science | pH range, microbial species, biochemical reactions |
| **Level 2 — Practitioner consensus** | Well-documented baker community standards, professional school curricula | Fermentation temperatures, hydration ranges, timing |
| **Level 3 — Author-derived** | Author's 847-bake log, 40-workshop data set | Workshop failure rate statistics, student success rate data |

When a claim has conflicting values from different levels, the higher-level source takes precedence unless the author can document a specific exception from their data with statistical justification.

## Part III: Canon Conflict Resolution Protocol

If a new source contradicts a registered canonical value:

1. Note the conflict in this document with date and source citation
2. Do NOT change the canonical value until resolution
3. Resolve by: comparing source authority levels; if same level, defer to most recent research
4. Update canonical value with new source reference
5. Run a full CFR audit pass on the manuscript to update all instances

---

## Part IV: Usage in Manuscript

When generating text with numeric claims:
- Check this registry before writing
- Use exact canonical values (no rounding, no narrowing)
- Cite the claim ID in draft comments if helpful: `<!-- CFR P-004: bulk ferment 4–6 hours at 75°F -->`
- Flag any claim in the manuscript that does not appear in this registry (it may need to be added)
